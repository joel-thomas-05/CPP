# HW 02 Problems

- problem1.cpp - Categorizes numerical score into letter grades (A, B, C, D, E, F) using if-else statements
- problem2.cpp - Prints "Hello World!" 100 times using:
(a) for loop
(b) while loop
(c) do-while loop
- problem3.cpp - Determines the most frequently occuring character in a given string

# Compile Example

g++ problem1.cpp -o P1

Replace filename as needed:
g++ problemX.cpp -o problemX

# Run

./P1
./P2
./P3

# Notes:

- Problem 1 requires one integer (student score)
- Problem 2 prints 300 total lines (100 per loop)
- Problem 3 requires a single string input (no spaces)
