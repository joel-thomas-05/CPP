# HW 03 Problems

- problem1.cpp - Declares an array {1,2,3,4,5} and swaps the 2nd and 4th elements using pointers.
- problem2.cpp - Takes two integer inputs from the user and computes their sum using pointers.
- problem3.cpp - Reverses an array using a pointer-based approach.
- problem4.cpp - Defines a function `swapNumbers(int* a, int* b)` that swaps two numbers using pointers and calls the function in `main()` with user-inputted values.

# Compile Example

g++ problem1.cpp -o P1
