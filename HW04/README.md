# HW 02 Problems

- problem1.cpp – Defines a `Car` class with private members `brand` and `year`. Uses `setBrand()` and `setYear()` to assign values and prints the car brand and year.
- problem2.cpp – Defines a `Book` class with private members `title` and `pages`. Uses `setTitle()` and `setPages()` to assign values and prints the book title and page count.
- problem3.cpp – Demonstrates inheritance with a base class `Animal` and derived class `Dog`. Uses `setSpecies()` and `setBreed()` to assign values and prints the species and breed.

# Compile Example
